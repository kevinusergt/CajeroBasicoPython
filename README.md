# CajeroBasicoPython
Simulador de cajero automatico

# Funciones principales
Actualmente el simulador cuenta
con las siguentes opciones:

*Consultar saldo
*Retirar saldo
*Depositar al saldo

# ¿Cómo usarlo o descargarlo?

Al entrar a este repositorio..
copia: https://github.com/kevinusergt/CajeroBasicoPython.git
y luego en ordenador localizas cualquier carpeta..
Dentro de esa carpeta abres la terminal y escribes ....

``` git clone https://github.com/kevinusergt/CajeroBasicoPython.git ```
